# Аудит расходов

## Как использовать

Пользователь даёт путь к файлу (CSV или Excel) или URL, откуда файл можно скачать.

## Что делать

1. **Получить файл** — если пользователь дал URL, скачать файл и сохранить локально (сохранить оригинальное расширение — `.csv`, `.xlsx` и т.д.). Если дан путь — использовать файл как есть.
2. **Запустить `audit.py` с этим файлом**:
   ```
   python audit.py <путь_к_файлу>
   ```
   Скрипт сам определяет, какие столбцы содержат дату, категорию, сумму и описание — специальный формат или фиксированные названия колонок не требуются. Перед первым запуском убедиться, что установлены зависимости из `requirements.txt` (`pip install -r requirements.txt`).
3. **Прочитать результат** из `report.txt` (создаётся рядом, если не указан другой путь через `-o`).
4. **Написать краткую сводку для владельца магазина**: что не так с расходами, на что обратить внимание, где возможны проблемы (задвоенные платежи, аномально крупные суммы, отрицательные суммы, нехватка категорий и т.п.). Простым языком, без технических терминов (не "стандартное отклонение", а "сумма сильно выделяется на фоне остальных").
5. **Показать сводку пользователю**.

## Заметки

- Скрипт параметризован и работает с любым файлом расходов, а не только с конкретным примером — названия и порядок столбцов могут отличаться.
- Если в файле не удалось найти столбец с суммой — скрипт сообщит об этом и завершится с ошибкой; в этом случае нужно посмотреть на файл вручную и подсказать пользователю, что не так с форматом.
- Если не найден столбец категории или даты — соответствующие разделы отчёта (разбивка по категориям, часть проверки дубликатов) будут помечены как недоступные, а не выпадут молча.
